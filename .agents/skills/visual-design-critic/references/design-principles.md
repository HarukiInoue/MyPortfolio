# Visual Design Principles

## Photo First

The photograph should carry the page. UI should frame and organize, not decorate.

## Hierarchy

Users should recognize the main image, the portfolio identity, and the gallery path quickly.

## Restraint

Avoid visual effects that make the interface more memorable than the images.

## Rhythm

Use consistent spacing, image ratios, and text scale to create a quiet browsing cadence.

## Mobile Quality

Mobile is not a compressed desktop. It should keep images large, captions readable, and navigation easy to tap.

## Maintainability

Prefer shared tokens and reusable classes over one-off CSS.

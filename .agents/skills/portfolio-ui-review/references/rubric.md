# Portfolio UI Review Rubric

Use this rubric before editing and after each iteration.

## Required Scores

- Photo dominance: 9/10
- Visual hierarchy: 8/10
- Spacing and rhythm: 8/10
- Typography: 8/10
- Mobile usability: 8/10
- Implementation maintainability: 8/10

## Review Prompts

- Does the first viewport make photography the main subject?
- Are decorative elements quieter than the photographs?
- Is the gallery easy to scan without feeling like a dense thumbnail sheet?
- Do captions help orientation without becoming the main content?
- Does the mobile layout keep the photos large and comfortable?
- Can the change be explained as one focused design improvement?

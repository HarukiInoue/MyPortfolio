# Gallery Patterns

## Recommended Pattern

Use a regular grid with occasional wide or tall images. This creates rhythm without the instability of arbitrary masonry.

## Desktop

- Four-column grids work well for broad selected-work pages.
- Use `grid-column: span 2` for strong horizontal images.
- Use `grid-row: span 2` for portraits when the surrounding rhythm supports it.
- Keep gaps consistent and avoid nested card treatments.

## Mobile

- Prefer a single column for image appreciation.
- Let captions sit below images.
- Avoid hover-only interaction cues.

## Interaction

- Clickable images should have a clear cursor or focus state.
- Hover effects should be subtle: contrast or slight tonal response is enough.
- Lightbox controls should be obvious and keyboard-friendly.

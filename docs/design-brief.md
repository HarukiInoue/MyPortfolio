# Design Brief

## Direction

Create a calm, editorial photography portfolio where images dominate the page and the interface stays quiet.

## Audience

Visitors should quickly understand the photographer's style, browse selected work comfortably, and find basic context without leaving the gallery flow.

## Visual Tone

- Gallery-like
- Restrained
- Spacious
- Photo-first
- Minimal but not empty
- Interactive, but only when interaction reveals the photographic sequence or the grid structure

## Things To Avoid

- Decorative gradients or ornamental backgrounds
- Heavy cards, borders, shadows, and hover effects
- Dense thumbnail walls that make images feel disposable
- Typography that competes with the photographs
- Mobile layouts that make the first photograph feel secondary
- Motion that feels like decoration rather than a transition from immersion to order

## Interactive Direction

The preferred interaction model is:

1. The page opens with one photograph large enough to feel immersive.
2. As the visitor scrolls, that photograph settles into the existing Swiss grid.
3. The gallery keeps a quiet current-photo index/title so the visitor understands their place in the sequence.
4. Reduced-motion users receive the settled layout without losing content.

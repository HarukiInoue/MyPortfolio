# UI Evaluation Rubric

Score each dimension from 1 to 10.

## Photo Dominance

Photos should be visually dominant and large enough to appreciate.

Target: 9/10

## Visual Hierarchy

Within 3 seconds, users should understand the portfolio identity, main photo area, and gallery flow.

Target: 8/10

## Spacing And Rhythm

Spacing should feel consistent, editorial, and calm without wasting too much screen space.

Target: 8/10

## Typography

Type should be restrained, readable, and supportive of the imagery.

Target: 8/10

## Gallery Layout

The gallery should handle mixed aspect ratios intentionally and avoid arbitrary visual noise.

Target: 8/10

## Mobile Usability

Mobile should preserve image presence, comfortable taps, readable captions, and clear navigation.

Target: 8/10

## Maintainability

Changes should reuse existing components, tokens, and classes where possible.

Target: 8/10

## Interactive Photo Reveal

The opening viewport should make the photograph feel immersive before the page settles into interface structure.

Target: 8/10

## Scroll-To-Grid Settlement

Scroll-linked motion should visibly resolve the large photograph into the established grid system.

Target: 8/10

## Gallery Orientation

Long-gallery browsing should provide quiet orientation through current image number/title without competing with the image.

Target: 8/10

## Motion Restraint And Accessibility

Motion should be limited, purposeful, performant, and disabled or simplified for `prefers-reduced-motion`.

Target: 8/10
